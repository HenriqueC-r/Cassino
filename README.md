# 🎰 Cassino em Python

Este é meu segundo projeto em Python.

O programa simula uma máquina caça-níquel no terminal, onde o jogador pode girar as frutas e tentar acertar três iguais para ganhar o JACKPOT.

---

## 🚀 Tecnologias utilizadas

- Python 3
- Biblioteca `random`
- Biblioteca `time`

---

## 🎮 Como funciona

- O jogador pressiona ENTER para girar.
- As frutas rodam de forma animada no terminal.
- Se as três frutas forem iguais → 🎉 JACKPOT!
- Caso contrário → perdeu.

---

## ▶️ Como executar

No terminal:

```bash
python3 cassino.py
